Personal Automation Tool

A configurable file organizer (Downloads/Desktop “inbox” sorter) with dry-run, duplicate strategies, rule priority/regex matching, and undo via a move ledger.

Where is the GUI?

The interactive GUI is a Streamlit app in this repo: streamlit_app.py. GitHub Pages cannot run Python, so this page is a static landing page. 

pip install -r requirements-ui.txt
streamlit run streamlit_app.py

For a hosted GUI, deploy streamlit_app.py to Streamlit Community Cloud, Render, Fly.io, etc.

CLI quick start

Run in dry-run first, then execute moves (and optionally record a ledger for undo).

python -m src.main --config config/rules.yaml --dry-run
python -m src.main --config config/rules.yaml --ledger-file logs/move_ledger.jsonl
python -m src.main --undo-last --ledger-file logs/move_ledger.jsonl

Features

  • Rules match by extensions, pattern, or regex
  • Conflict resolution via rule priority (highest wins)
  • Duplicate handling: skip, rename, overwrite
  • Undo support using a JSONL move ledger (--undo-last)
  • Optional cleanup: delete empty directories
